Q.16 What are the advantages and disadvantages of using external style sheets?

Advantages:

Reusability:
Styles defined in an external CSS file can be applied across multiple HTML files, ensuring consistency in design throughout a website.
Maintenance:
Centralizing styles in a separate file makes it easier to manage and update styles universally without the need to modify each HTML file individually.
Faster Loading:
External CSS files can be cached by the browser after the initial visit, resulting in quicker loading times for subsequent visits to pages that use the same stylesheet.

Disadvantages:

Network Dependency:
If the external stylesheet fails to load due to network issues or errors, it can affect the appearance and layout of the web page.
Increased HTTP Requests:
Loading external stylesheets requires an additional HTTP request, which might marginally increase the webpage's loading time, especially if there are multiple stylesheets.
