Q.10 How can you convert the string of any base to an integer in JavaScript?

In JavaScript, you can convert a string representing a number in any base to an integer using the "parseInt()" function. The parseInt() function takes two parameters: the string to parse and the base of the numeral system
Ensure that the string represents a valid number in the specified base; otherwise, parseInt() might return unexpected results. Also the output will always be in base 10 (decimal), regardless of the input base.
