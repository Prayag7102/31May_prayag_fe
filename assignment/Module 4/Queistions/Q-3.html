Q.3 What is the difference between CSS2 and CSS3 ?

CSS3 is an enhanced version of CSS that introduced several new features and improvements over CSS2. Some of the key differences include:

1. Selectors:-
CSS3 introduced new and more powerful selectors that allow developers to target specific elements more precisely. Selectors like attribute selectors, nth-child, and :not() were added in CSS3.
2. Flexbox and Grid:-
CSS3 introduced flexible box (flexbox) and grid layout systems, which significantly enhance the control and ease of creating complex layouts compared to the limitations of CSS2
3. Multiple Backgrounds:-
CSS3 allows for the use of multiple background images on an element, giving designers more flexibility in creating visually rich layouts.
4. Custom Fonts:-
CSS3 introduced @font-face, enabling the use of custom fonts on websites, expanding typographic possibilities beyond the standard web-safe fonts available in CSS2.
5. Transforms and Transitions:-
CSS3 introduced properties like transform and transition, allowing developers to perform various transformations (such as rotations, scaling, skewing) and create smooth transitions between CSS property values.
6. Text Effects:-
CSS3 includes properties for text effects, such as text-shadow, which enables the creation of shadow effects around text, and text-overflow, allowing for better control over text when it overflows its container.
7. Animations and Transitions:-
CSS3 brought inbuilt support for animations and transitions, allowing developers to create dynamic and engaging effects without relying on JavaScript.
