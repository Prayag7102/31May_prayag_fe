Q.9 Which property controls the image scroll in the background?

The property that controls the scrolling behavior of a background image in CSS is called background-attachment. This property specifies whether the background image should scroll with the content or remain fixed within the viewport as the content scrolls.
The background-attachment property can take the following values:

1.scroll: This is a default value.

2.fixed: The background image remains fixed in place within the viewport. Even when the user scrolls the webpage, the background image stays in its specified position.
