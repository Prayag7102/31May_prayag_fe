Q.5 What are undeclared and undefined variables?

Undeclared Variables:
These are variables that haven't been declared using var, let, or const before being used in the code. When you attempt to access a variable that hasn't been declared, you'll typically encounter a reference error.
Undefined Variables:
These are variables that have been declared but not assigned a value explicitly. In JavaScript, when you declare a variable without assigning any value to it, the variable holds the value undefined by default until a value is assigned to it.
