Q.2 What are the disadvantages of CSS ?

1. Connected to HTML:-
If you change the structure of your website, you might have to tweak your CSS too. It's like a matching outfit - if you change one piece, you might need to adjust the others to keep everything looking good together.
2. Layout Control:-
Sometimes, especially with older browsers or specific layout needs, CSS might not give you all the control you want. It can be a bit limiting in certain situations.
3. Browser Compatibility:
Sometimes, what looks great in one browser might appear wonky in another. Making sure your site looks good across all browsers can be a bit of a headache.
4. Tricky Complex Designs:-
If you're going for a super fancy layout or design, CSS might not always make it easy. You might need some workarounds or clever tricks to get things looking just right.
5. Dynamic Limits:-
CSS is great for styling, but it's not as good at making really interactive or dynamic stuff like JavaScript can. Sometimes it falls a bit short in that department.
