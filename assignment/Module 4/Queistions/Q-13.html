Q.13 What are the ways to integrate CSS as a web page?

There are some ways to integrate CSS into a web page:

Inline CSS:-
You can apply CSS styles directly to HTML elements using the style attribute. This method is useful for applying specific styles to individual elements but is less recommended for large-scale styling due to maintenance issues.
Internal CSS:
< style > Use the tag within the <head> section of an HTML document to define CSS styles for that specific page. This method allows for better organization than inline styles but might not be easily reusable across multiple pages.
External CSS:
Create a separate CSS file and link it to your HTML document using the <link> tag. This method promotes reusability, consistency across multiple pages, and easier maintenance.
