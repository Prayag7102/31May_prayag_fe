Q.12 What are all the types of Pop up boxes available in JavaScript?

JavaScript provides three types of pop-up boxes that developers can use to interact with users:
1. Alert Box:
Displays a message to the user in a dialog box and typically contains an OK button.
Syntax:- alert("This is an alert message");
2. Confirm Box:
Presents a message and gives the user the option to choose between "OK" and "Cancel". It returns a boolean value (true for OK and false for Cancel).

Syntax:- let result = confirm("Do you want to proceed?");
Prompt Box:
Displays a message along with an input field to receive user input. It returns the text entered by the user or null if canceled.
Syntax:- let userInput = prompt("Please enter your name:", "John Doe");
