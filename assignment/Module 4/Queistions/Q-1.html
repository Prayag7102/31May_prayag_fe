Q.1 What are the benefits using CSS ?

There are many benefits using CSS like:-

1.Enhanced User Experience :-
CSS allows for creative and interactive design elements that enhance user experience, such as animations, transitions, and responsive layouts.
2.Responsive Design:-
With CSS, you can create responsive designs that adapt to different screen sizes and devices. Media queries and flexible layout options enable websites to look good on desktops, tablets, and mobile devices.
3.Faster Loading Times :-
External CSS files can be cached by the browser, allowing subsequent pages to load faster after the initial visit. This optimization helps improve the overall performance of a website.
4.SEO Benefits :-
Properly structured HTML combined with well-applied CSS can enhance a website's search engine optimization (SEO) by making it easier for search engine.
Platform Consistency:-
CSS helps maintain a consistent look and feel across different browsers and platforms, ensuring that your website appears similar regardless of the user's device or browser choice.
