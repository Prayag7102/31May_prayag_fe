Q.4 Name a few CSS style components

Here are a few CSS style components :-

1. Color:-
CSS allows you to set colors for text, backgrounds, borders, and other elements using properties like color, background-color, and border-color.
2. Typography:
Properties like font-family, font-size, font-weight, and line-height are used to control the appearance of text, including the font style, size, weight, and spacing.
3. Layout:
CSS offers various properties for layout control, including display (to define how elements are displayed), position (to position elements), float (for floating elements), and newer layout systems like flexbox and grid.
4. Responsive Design:-
CSS includes media queries to create responsive designs, allowing you to adjust styles based on different device characteristics, such as screen width, height, orientation, or resolution.
5. Animations and Transitions:-
CSS properties like transition, animation, and their associated properties (animation-duration, animation-timing-function, etc.) enable the creation of animations and transitions for elements on a webpage.
6. Transforms:-
Properties like transform allow you to modify the appearance of elements by applying transformations such as rotations, scaling, skewing, and translations.
7. Backgrounds:-
CSS provides properties like background-image, background-size, background-position, and background-repeat to control the background appearance of elements.
