JavaScript offers several looping structures to iterate through data or execute a block of code repeatedly.
for loop:
A versatile loop that executes a specified block of code a specified number of times.
Syntax:- for (initialization; condition; increment/decrement) { // code to be executed }
while loop:
Executes a block of code as long as the specified condition is true.
Syntax:- while (condition) { // code to be executed }
do...while loop:
Similar to the while loop, but it executes the code block at least once before checking the condition.
Syntax:- do { // code to be executed } while (condition);
for...in loop:
Iterates over the enumerable properties of an object.
Syntax:- for (variable in object) { // code to be executed }
for...of loop:
Introduced in ES6, it iterates over iterable objects like arrays, strings, maps, etc.
Syntax:- for (variable of iterable) { // code to be executed }
