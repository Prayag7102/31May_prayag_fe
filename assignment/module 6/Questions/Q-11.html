Q.11 The delete operator in JavaScript is used to remove a property from an object or to remove an element from an array. Its function depends on what you're trying to delete:

From Objects:
const obj = { a: 1, b: 2, c: 3 }; // Delete a property from the object delete obj.a; // Now obj will be { b: 2, c: 3 }
Deleting Array Elements:
const arr = [1, 2, 3, 4, 5]; // Delete an element from the array delete arr[2]; // Now arr will be [1, 2, , 4, 5] // Note: The array length doesn't change, but the deleted element becomes undefined.
