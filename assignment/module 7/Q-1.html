Q.1 What is jQuery?

jQuery is a popular, lightweight JavaScript library designed to simplify the client-side scripting of HTML. It was created to ease the complexities of JavaScript and to make it easier to navigate and manipulate HTML documents, handle events, perform animations, and simplify AJAX interactions.
DOM Manipulation:
-> jQuery provides simplified methods to select, traverse, and manipulate DOM elements. It allows developers to easily change content, style, and structure of web pages.
Event Handling:
-> jQuery simplifies the process of attaching event handlers to elements and managing their behavior when events occur, such as clicks, keyboard inputs, and more.
Animations:
It offers built-in methods for creating animations and effects on web pages, allowing developers to create visually appealing transitions and movements.
